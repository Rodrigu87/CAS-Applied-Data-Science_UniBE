# CAS-Applied-Data-Science_UniBE
This repo is a collection of learning materials and projects showcasing my learning journey in data science. The repo is structured into six modules, with each module including both the learning materials and its corresponding project. It also contains my final project for the CAS

# About the CAS Applied Data Science ( CAS ADS)
The CAS in Applied Data Science is a professional certificate program at the University of Bern equipping participants with practical skills and tools for real-world data analysis and machine-learning applications.
