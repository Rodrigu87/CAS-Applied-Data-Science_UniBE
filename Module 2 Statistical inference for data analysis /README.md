# About Module 2. Statistical inference for data analysis

This module provides a rigorous introduction to statistical inference for data analysis. It covers the principles of estimation, hypothesis testing, and uncertainty quantification, enabling participants to draw valid conclusions from data and support decision-making with statistically sound evidence. 

# Learning objectives
+ Understand the fundamental concepts of statistical inference, including sampling, estimation, and uncertainty
+ Apply point estimation and confidence interval techniques to quantify parameters from data
+ Formulate and conduct hypothesis tests to support data-driven decisions.
+ Implement and interpret statistical inference workflows in Python for real-world data analysis.
