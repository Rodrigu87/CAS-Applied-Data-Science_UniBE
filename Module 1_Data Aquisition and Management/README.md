This module introduces the foundations of working with data in applied data science. It focuses on acquiring data from diverse sources, managing structured and unstructured datasets, and preparing data for analysis and modeling.
Learning Goals
Understand key concepts in data acquisition and data pipelines
Work with common data formats and sources (CSV, JSON, APIs, databases)
Apply best practices in data cleaning and preprocessing
Manage datasets using Python and essential data-science libraries
Document workflows and ensure reproducibility
Contents of This Folder
Learning materials: notes, exercises, and scripts used throughout the module
Module project: a hands-on assignment applying the concepts of data acquisition, cleaning, and management
Tools & Libraries
Python
Pandas
NumPy
Requests / API clients
Jupyter Notebooks
Objective
By the end of this module, learners are able to acquire real-world data, clean it effectively, structure it for analysis, and document the entire pipeline in a reproducible workflow.
